# HR Analytics Dashboard
**Tools:** Power BI

## Overview
Employee attrition analysis for a 1,470-person workforce.
Built an interactive Power BI dashboard to identify key drivers
of attrition and provide actionable HR recommendations.

## Key Findings
- Overall attrition rate: **16.1%** (237 out of 1,470 employees)
- **69% of attrition** comes from the under-$5K salary band
- Lab Technicians (62) and Sales Executives (57) — highest attrition roles
- Age group 26–35 has the highest attrition count of **116 employees**

## Dashboard Features
- 6 KPI cards: Total employees, Attrition, Rate, Avg Salary, Avg Age, Avg Tenure
- Attrition breakdown by: Education, Age Group, Salary Slab,
  Years in Company, Job Role
- Interactive slicers for Department, Gender and Education
